# Glitza_Nails
Glitza Nails is a modern and responsive e-commerce website built for a nail studio specializing in premium Press-On Nails and Professional Nail Extension Services. The platform provides customers with an easy and elegant way to browse products, customize nail designs, and book nail services online.

✨ Features
🛍️ Browse and order premium Press-On Nail sets.
🎨 Customize Press-On Nails according to customer preferences.
📏 Available in multiple sizes (S, M, L).
💅 Book Temporary Nail Extensions for hands and toes.
💎 Book Soft Gel Nail Extensions for hands and toes.
📱 Fully responsive design for desktop, tablet, and mobile devices.
🖼️ Beautiful product gallery with detailed service information.
🛒 Simple and user-friendly shopping experience.
📞 Easy contact and appointment booking.
🚀 Purpose

Glitza Nails is designed to provide a seamless online shopping and booking experience for nail enthusiasts. Customers can explore ready-made nail collections, request personalized designs, and schedule professional nail extension services—all from one platform.
